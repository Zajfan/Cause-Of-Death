# Cause of Death

A Python desktop/localhost crime-solving prototype.

## Prototype features

- Case list
- Evidence viewer
- Suspect panel
- Notes area
- Accusation screen

## How to run

```bash
python3 app.py
```

If you are in a headless environment, open the local web app instead:

```bash
python3 app.py
# then visit http://127.0.0.1:8000
```
