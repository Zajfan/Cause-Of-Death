# Cause of Death — Roadmap

## Phase 1: Foundation

- Define the first playable case
- Build the case file UI
- Build the evidence viewer
- Add suspect profiles
- Add a notes panel
- Support one complete accusation flow

## Phase 2: Evidence handling

- Add image viewer
- Add audio player
- Add video player
- Add document viewer
- Add evidence tagging and filtering
- Add timeline tools

## Phase 3: First mystery set

- Create 3 to 5 handcrafted murder cases
- Mix different case styles
- Add hints
- Add scoring
- Add case results and debrief screens

## Phase 4: Deduction depth

- Add contradiction highlighting
- Add deduction board / theory map
- Add suspect relationship tracking
- Add stronger false-alibi logic
- Add case difficulty levels

## Phase 5: Content tools

- Add case editor
- Add content import/export
- Add templates for suspects, clues, and scenes
- Add support for community-made cases

## Phase 6: Polishing

- Improve UI flow
- Add sound design
- Add animation and transitions
- Add accessibility options
- Add save/load and case history

## First build target

The first milestone should be a single fully playable murder case with:

- 1 victim
- 4 suspects
- 12 to 20 evidence items
- 1 hidden twist
- 1 correct final accusation
